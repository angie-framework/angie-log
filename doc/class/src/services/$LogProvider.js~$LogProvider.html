    $LogProvider

    $LogProvider is the only class in the Angie Logging module. This module allows for logging to the terminal (with a little flavor).

    This can also be referenced as Log

    Example:

    new $LogProvider();

    public logger(out: string): * since 0.0.2 source

    $LogProvider.logger will add a log statement for each call that is made. It pushes messages to an asynchronous queue, which will execute as messages are added.

    Params:

    NameTypeAttributeDescription
    out string

    The message to add to the log

    Return:

    *

    Example:

    new $LogProvider(output.log, true, 'DEBUG', false).logger('test');
